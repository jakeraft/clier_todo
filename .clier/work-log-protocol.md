# Work Log Protocol

Use `clier run note` to record work log entries.
Record a note when you start work, complete work, hit a blocker, or hand off context.
If someone asks you to write, record, post, or leave a note, use `clier run note`.
Keep each note brief and factual.
When a note mentions concrete work, include direct references such as file paths, command names, resource names, or URLs whenever they would help someone continue from the note later.

```bash
clier run note <<'EOF'
<content>
EOF
```
